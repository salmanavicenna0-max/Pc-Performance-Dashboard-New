# smonitor

A new Flutter project.

System minimalist dashboard pc and android.
Cara Menjalankan Server: perintah pip install fastapi uvicorn psutil wmi ping3 lalu jalankan Start_Dashboard.bat.
Cara Menjalankan Android: penjelasan singkat cara menyambungkan IP Address di file main.dart.

System ini saya buat untuk keperluan saya pribadi, karna ingin memonitoring pc saya saat sedang gaming atau kerja.
